# MediQ — Smart Hospital Queue

MediQ is an interactive hospital queue management system built with HTML, CSS, and JavaScript. It automates patient token generation, provides a live queue display, and dynamically manages priority and emergency cases, ensuring smoother patient flow and reduced waiting times.

The interface features a hospital-friendly card layout, color-coded indicators, and a modern blue theme for clarity and usability.

Features
1. Real-time Now Serving Updates: Countdown timers for the current patient being served.
2. Animated Queue Progress: Visual progress bars showing estimated wait times for each patient.
3. Interactive Controls: Call, promote, or remove patients directly from the live queue.
4. Priority & Emergency Handling: Emergency and priority patients are automatically sorted to the front of the queue.
5. Patient History: All served or removed patients are recorded in a history page.
6. Daily Totals: Automatic syncing of the daily patient count with history.
7. Search Functionality: Search tokens or patient names for quick access.
8. Clear Queue / History: Reset the queue and history with a single click.


Technologies Used
1. HTML5 — For structure and layout
2. CSS3 — For modern, responsive styling and theming
3. JavaScript (ES6) — For frontend logic, real-time updates, and local data storage
4. LocalStorage — To persist queue and history data across sessions

Potential Enhancements
1. Backend Integration: Store queue data in a database for multi-user support.
2. IoT Display Boards: Show live queue on hospital monitors.
3. Notifications: Send SMS/email alerts when patient tokens are called.
4. Mobile App: Make MediQ accessible on smartphones for patients and staff.
